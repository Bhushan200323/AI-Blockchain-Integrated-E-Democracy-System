# 📖 Overview

AI-Blockchain Integrated E-Democracy System is a secure, transparent, and intelligent digital governance platform that combines Blockchain Technology, Artificial Intelligence, and Cloud Computing to modernize elections and improve democratic accountability.

The platform enables secure online voting, blockchain-backed vote verification, AI-powered election insights, representative accountability tracking, and public issue management while ensuring transparency and trust.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Bhushan200323/AI-Blockchain-Integrated-E-Democracy-System.git
```

Go to the project directory

```bash
cd AI-Blockchain-Integrated-E-Democracy-System
```

Create virtual environment

```bash
python -m venv venv
```

Activate virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file and configure the required environment variables.

Run the application

```bash
python app.py
```

or

```bash
flask run
```
